    GreasePencil: A Caffeinated WindowsScript Framework

    Project maintained by UniDyne Hosted on GitHub Pages — Theme by mattgraham

    What Is It?

    GreasePencil adds the flavor of browser-based JavaScript frameworks to JScript and WindowsScript. Additionally, GreasePencil brings a library of functionality specific to the everyday needs of script writers in the world of business. This includes database and reporting functionality as well as integration with Microsoft Office.

    The Name

    GreasePencil was named such because grease pencils are what one would normally use to write on glass. The phrase WindowsScript makes me think of this for some reason. Just as grease pencils can be used to write script on windows, GreasePencil can be used to enhance WindowsScript.

    Why Not Node.js? Or PowerShell?

    This project is not an attempt to create a generic JavaScript environment but is an effort to create a framework for business-related tasks in a Windows environment. As such, Windows already includes JScript as part of Windows Scripting Host. The roots of this project predate both Node and PowerShell.

    Authors and Contributors

    GreasePencil is currently developed and maintained by Darien O. Brown (@unidyne). Some of the JScript language extensions used in this project were derived from early versions of Prototype and/or MooTools.
